# 文字全明星

一个基于 **HTML / CSS / JavaScript** 实现的汉字塔防小游戏。

玩家通过征兵获得汉字，把汉字放置到棋盘上进行防守。相同汉字可以升级，不同汉字可以组合成特殊武将。敌人会沿路线进攻阿斗，守住阿斗即可通关。

在线游玩地址：

https://strelizia-zerotwo.github.io/hanzi_quanmingxing_defense/

---

## 游戏简介

《文字全明星》是一款汉字合成塔防游戏，核心玩法包括：

- 汉字布阵
- 单字升级
- 多字合成武将
- Roguelike 强化选择
- Boss 关卡
- 无尽模式
- 人机匹配模式

玩家需要合理安排汉字单位的位置，不断升级阵容，抵挡一波又一波敌人的进攻。

---

## 游戏模式

目前包含以下模式：

### 荒野古道

基础关卡，适合熟悉游戏机制。

### 火烧赤壁

路线更加复杂，敌人压力更高。

### 无尽模式

敌人会不断增强，Boss 会循环出现，挑战玩家阵容上限。

### 匹配模式

人机对战模式。

双方各有一张棋盘，同时防守同样节奏的敌人。谁的阿斗先被攻破，谁就失败。

人机会自动：

- 征兵
- 布阵
- 升级
- 合成武将
- 选择强化
- 防守敌人

---

## 基础玩法

### 征兵

消耗馒头进行征兵，获得新的汉字单位。

### 布阵

将汉字拖动到白色可部署格子中。

### 升级

相同汉字、相同等级可以合成更高等级。

### 合成武将

部分汉字可以组合成特殊武将。

例如：

- 黄 + 忠 = 黄忠
- 科 + 比 = 科比
- 马 + 超 = 马超
- 马 + 云 = 马云
- 赵 + 云 = 赵云
- 张 + 雪 + 峰 = 张雪峰

---

## 武将介绍

### 黄忠

远程武将，攻击距离远。

技能：**我的大炮打不打**

释放技能时进行大范围攻击。

### 科比

篮球主题武将。

技能：**曼巴 out**

释放技能时召唤直升机攻击敌人。

### 马超

近战突击型武将。

使用长枪攻击敌人，攻击时枪会指向目标。

### 马云

辅助型武将。

可以产生馒头资源，并使用金钱眩晕敌人。

### 赵云

雷电型武将。

攻击会附带雷电伤害，技能期间可以造成额外连锁雷电伤害。

### 张雪峰

冰系控制型武将。

普攻发射冰块，造成伤害并减速敌人。

技能：**你跑不过我你信吗**

释放技能时，张雪峰会冲向距离阿斗最近的敌人，将敌人整体击退，并造成短暂眩晕。

## 强化系统

每波结束后，玩家可以选择一个强化。

强化可能影响：

- 单位伤害
- 单位攻速
- 单位攻击范围
- 武将技能冷却
- 馒头生成速度
- 兵营容量
- 阿斗血量
- 斧头系统
- 特定武将刷新概率

匹配模式中，人机也会随机选择强化。
